Pluely — customized

A Windows AI meeting assistant: live transcription, screenshots, and instant answers. Self-contained — speech-to-text is bundled, no setup required.

⬇ Download for Windows
v0.1.9 · ~202 MB · Windows 10 / 11 (64-bit) · installer (.exe)

Localno account

Fully offline. Bundled Whisper speech-to-text + the minicpm-v vision model (pulled at install, ~5.5 GB) so screenshots work too. No API key, no login.

Claudesmarter

Routes answers through your local claude CLI. Needs a one-time claude login and a Claude subscription. Speech-to-text still bundled.

  • Pick Local or Claude during installation — switchable later in Settings.
  • The app starts and stops its bundled helper servers automatically.
  • Unsigned build: Windows SmartScreen will warn — click “More info” → “Run anyway”.
